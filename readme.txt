Языки подключаються в файле assets>i18n.js
Файлы с переводами находяться в файле public>locales>common.json
Для добавление переменых с переводом на страницы нужно добавить файл assets>useLanguageSwitch.js и вызвать const { t, locale, locales, handleClick } = useLanguageSwitch();
Далее в файле где нужно добавить перевод надо добавить {t('сюда название из json файла')}
